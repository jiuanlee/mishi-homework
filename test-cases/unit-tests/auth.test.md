# 单元测试用例 - 认证模块

## UT-001: 用户注册 - 成功场景

| 项目 | 内容 |
|------|------|
| **用例编号** | UT-001 |
| **测试类型** | 单元测试 |
| **优先级** | P0 |
| **测试目标** | 验证用户注册功能在正常输入下能成功创建用户 |
| **前置条件** | 数据库连接正常，用户名不存在 |
| **测试步骤** | 1. 调用注册接口传入有效用户名、密码、邮箱<br>2. 验证返回状态码为 201<br>3. 验证返回包含 success: true<br>4. 验证返回包含 userId |
| **预期结果** | 注册成功，返回用户 ID |
| **测试数据** | `{username: "testuser", password: "123456", email: "test@example.com", role: "student"}` |
| **执行状态** | TODO |
