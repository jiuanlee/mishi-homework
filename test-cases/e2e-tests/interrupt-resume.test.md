# 端到端测试用例 - 中断续做场景

## 说明

测试学生在答题过程中遇到各种中断情况后，能否正确恢复答题进度。

---

## E2E-201: 小程序后台切换

| 项目 | 内容 |
|------|------|
| **用例编号** | E2E-201 |
| **流程名称** | 切换到后台后返回 |
| **优先级** | P0 |
| **前置条件** | - 学生正在答题 (已完成 3 题) |
| **测试步骤** | 1. 学生答题到第 4 题<br>2. 按 Home 键切换到后台<br>3. 等待 1 分钟<br>4. 重新打开小程序 |
| **预期结果** | - 答题进度自动保存<br>- 返回后显示"继续答题"<br>- 前 3 题答案保留<br>- 从第 4 题继续<br>- 倒计时继续运行 |
| **验证点** | - 本地存储进度<br>- 服务端同步状态 |
| **执行状态** | TODO |
