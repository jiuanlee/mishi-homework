## 超时时间配置

| 场景 | 超时时间 | 处理方式 |
|------|----------|----------|
| API 请求 | 30 秒 | 重试 3 次 |
| 文件上传 | 120 秒 | 断点续传 |
| 数据库查询 | 10 秒 | 返回错误 |
| Redis 操作 | 5 秒 | 降级处理 |
| 会话有效期 | 24 小时 | 重新登录 |
| 作业总时长 | 可配置 | 自动提交 |
| 心跳超时 | 5 分钟 | 保存进度 |

| 错误码 | 说明 | 前端处理 |
|--------|------|----------|

| TIMEOUT_002 | 上传超时 | 支持续传 |
| TIMEOUT_003 | 会话过期 | 跳转登录 |
| TIMEOUT_004 | 作业超时 | 自动提交 |
| TIMEOUT_005 | 服务不可用 | 稍后重试 |
