| 等级 | 说明 | 响应时间 |

| 🔴 P0 | 严重 - 系统崩溃、数据丢失、核心功能不可用 | 立即修复 |

## Bug 生命周期

```
新建 → 确认 → 分配 → 修复中 → 已修复 → 验证 → 关闭
         ↓                    ↓
       拒绝 ←-------------- 未通过
```

### 状态说明

| 状态 | 说明 |
|------|------|
| 新建 | 测试人员提交的新 Bug |
| 确认 | 开发负责人确认是 Bug |
| 分配 | 分配给具体开发人员 |
| 修复中 | 开发人员正在修复 |
| 已修复 | 开发完成，待验证 |
| 验证 | 测试人员验证修复 |
| 关闭 | 验证通过，Bug 关闭 |
| 拒绝 | 不是 Bug 或无法重现 |
| 未通过 | 修复不彻底，重新打开 |
