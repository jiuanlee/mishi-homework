# 学生端 - 微信小程序

## 快速开始

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)

2. 打开微信开发者工具，导入项目

3. 填写 AppID（测试可选择测试账号）

4. 编译运行

## 目录结构

```
.
├── pages/
│   ├── login/      # 登录页面
│   ├── homework/   # 作业列表
│   ├── game/       # 游戏化做题
│   └── record/     # 学习记录
├── components/     # 小程序组件
├── utils/          # 工具函数
├── assets/         # 静态资源
├── app.js          # 小程序入口
├── app.json        # 小程序配置
└── app.wxss        # 全局样式
```

## 页面列表

1. login - 登录/注册（班级码）
2. homework - 作业列表
3. game - 游戏化做题界面
4. record - 作业记录页面

## 核心功能

- 班级码登录
- 作业列表展示
- 密室逃脱模式
- 剧本杀模式
- 线索解锁动画
- 本地缓存（中断续做）

## 开发规范

- 遵循微信小程序开发规范
- 使用 WXSS（兼容 SCSS 语法）
- 组件化开发
- 适配不同屏幕尺寸

## 注意事项

- 需要在微信公众平台配置服务器域名
- 提交审核前确保符合小程序内容规范
- 测试时使用测试账号即可
