## 🎮 游戏简介

欢迎来到贪吃蛇游戏！这是一款经典又令人上瘾的休闲游戏。

**游戏目标**：控制小蛇吃掉食物，让自己变得越来越长，同时避免撞墙或撞到自己，争取获得最高分！

**适合人群**：所有年龄段的玩家，无论是想消磨时间还是挑战高分，都能找到乐趣。

## 🚀 开始游戏

### 第一步：打开游戏

根据你的设备，选择以下方式之一：

**电脑端**：
- 在浏览器中打开游戏页面
- 或双击 `index.html` 文件

**手机端**：
- 在浏览器中访问游戏网址
- 或扫描提供的二维码

### 第二步：选择难度

游戏开始前，你可以选择难度等级：

| 难度 | 图标 | 描述 | 推荐人群 |
|------|------|------|---------|
| 简单 | 🟢 | 蛇移动速度慢，适合新手 | 第一次玩的玩家 |
| 普通 | 🟡 | 正常速度，平衡体验 | 有经验的玩家 |
| 困难 | 🔴 | 速度快，挑战反应 | 想挑战自己的玩家 |
| 地狱 | 🟣 | 极速，极限挑战 | 高手专属 |

**建议**：新手从「简单」开始，熟悉后再挑战更高难度。

### 第三步：开始！

点击「开始游戏」按钮，游戏立即开始！

## 🎯 操作说明

### 电脑键盘操作

#### 方向控制

```
        ↑ (上箭头)
           或
           W
           |
← (左箭头)  |  → (右箭头)
    或      |     或
    A       |     D
            |
           ↓ (下箭头)
           或
           S
```

| 按键 | 功能 |
|------|------|
| `↑` 或 `W` | 向上移动 |
| `↓` 或 `S` | 向下移动 |
| `←` 或 `A` | 向左移动 |
| `→` 或 `D` | 向右移动 |

#### 功能按键

| 按键 | 功能 | 说明 |
|------|------|------|
| `空格` | 暂停/继续 | 游戏进行中按空格暂停，再按继续 |
| `R` | 重新开始 | 游戏结束后按 R 快速重开 |
| `M` | 静音 | 开启/关闭游戏音效 |
| `ESC` | 返回菜单 | 返回主菜单 |

### 手机触屏操作

#### 滑动控制

```
    向上滑动 → 蛇向上移动
    向下滑动 → 蛇向下移动
    向左滑动 → 蛇向左移动
    向右滑动 → 蛇向右移动
```

**操作提示**：
- 在游戏区域任意位置滑动手指
- 滑动方向决定蛇的移动方向
- 动作要干脆，避免误触

#### 触摸手势

| 手势 | 功能 |
|------|------|
| 双击屏幕 | 暂停/继续游戏 |
| 长按 2 秒 | 打开设置菜单 |
| 捏合手势 | 缩放游戏画面（如支持） |

## 📜 游戏规则

### 基本规则

1. **蛇会自动前进**
   - 蛇会一直朝着当前方向移动
   - 你需要提前规划转向时机

2. **吃掉食物**
   - 蛇头碰到食物时，蛇身增长一节
   - 同时获得分数奖励
   - 食物会随机出现在空白区域

3. **避免碰撞**
   - ❌ 撞到墙壁 = 游戏结束
   - ❌ 撞到自己的身体 = 游戏结束
   - ✅ 安全区域 = 继续游戏

### 游戏结束条件

游戏会在以下情况结束：

```
┌─────────────────────────────────────┐
│  ⚠️  游戏结束！                      │
│                                     │
│  原因：                             │
│  • 撞到墙壁                         │
│  • 撞到自己的身体                   │
│                                     │
│  最终得分：XXX                      │
│  最高记录：XXX                      │
│                                     │
│  [再来一局]  [返回菜单]             │
└─────────────────────────────────────┘
```

### 特殊规则

**不能 180 度掉头**
- 如果蛇正在向右移动，不能直接向左转
- 必须先向上或向下，再转向
- 这是为了防止蛇立即撞到自己

**蛇身增长机制**
- 每吃一个食物，蛇身增加一节
- 蛇身越长，转弯越需要技巧
- 后期需要更大的活动空间

```
┌─────────────────┐
│ →→→→→→→→→→→→→ │
│ ↑             ↓ │
│ ↑  逐渐向内    ↓ │
│ ↑    螺旋     ↓ │
│ ↑             ↓ │
│ ←←←←←←←←←←←←← │
└─────────────────┘
```

#### 技巧 3：预留退路

```
┌─────────────────┐
│    食物         │
│      ↓          │
│  →→→→→→→→→→    │
│             ↑   │
```

#### 技巧 4：蛇身管理

- 前期：尽量保持蛇身紧凑
- 中期：预留足够的转弯空间
- 后期：避免蛇身缠绕自己

### 连击系统

连续吃食物可以触发连击奖励：

| 连击数 | 倍数 | 每个食物得分 |
|--------|------|-------------|

| 4-6 个 | 1.5x | 15 分 |
| 7-9 个 | 2x | 20 分 |
| 10+ 个 | 3x | 30 分 |

**提示**：连击会随时间衰减，保持连续进食！

## ❓ 常见问题

### Q1: 游戏无法启动怎么办？

**解决方法**：
1. 检查浏览器是否为最新版本
2. 清除浏览器缓存后重试
3. 尝试更换其他浏览器（推荐 Chrome）
4. 检查是否启用了 JavaScript

### Q2: 按键没有反应？

**解决方法**：
1. 确保游戏窗口处于激活状态（点击游戏区域）
2. 检查是否有其他程序占用键盘
3. 尝试使用 WASD 替代方向键
4. 刷新页面重新开始

### Q3: 游戏太卡/不流畅？

**解决方法**：
1. 降低游戏难度（速度会变慢）
2. 关闭其他占用资源的程序
3. 检查网络连接（如为在线版本）
4. 尝试全屏模式

### Q4: 分数没有保存？

**可能原因**：
- 使用了浏览器隐私模式
- 浏览器禁用了 LocalStorage
- 清除了浏览器数据

**解决方法**：
- 退出隐私模式
- 在浏览器设置中允许网站存储数据
- 使用同一浏览器继续游戏

### Q5: 手机端操作不灵敏？

**解决方法**：
1. 确保手指干燥，避免误触
2. 在游戏区域内滑动，不要在边缘操作
3. 调整滑动速度，不要太快或太慢
4. 尝试横屏模式（如支持）

### Q6: 如何查看历史最高分？

**查看方法**：
- 游戏主菜单显示当前难度的最高分
- 点击「排行榜」查看详细记录
- 分数按难度分类保存

## 💡 小贴士

### 新手建议

1. **从简单难度开始**：熟悉游戏节奏
2. **不要贪心**：安全比分数更重要
3. **提前规划**：想好下一步再转向
4. **保持冷静**：越急越容易出错

### 进阶建议

1. **练习蛇身控制**：熟悉蛇的移动惯性
2. **学会利用边界**：墙壁可以是保护
3. **观察食物刷新**：预判下一个食物位置
4. **保持连击**：连击奖励很可观

### 高手建议

1. ** memorize 地图**：记住常用路线
2. **极限操作**：练习紧贴蛇身转弯
3. **心理战术**：预测自己的预判（认真脸）
4. **享受过程**：高分是副产品，快乐才是目的
