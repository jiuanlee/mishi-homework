# 🐍 贪吃蛇小游戏

> 经典复古游戏，重温童年乐趣

## 📖 项目介绍

贪吃蛇（Snake）是一款经典的街机游戏，玩家控制一条蛇在封闭区域内移动，通过吃掉食物来增长身体长度。随着蛇身变长，游戏难度逐渐增加，考验玩家的反应能力和策略规划。

本项目是一个现代化的贪吃蛇游戏实现，具有以下特点：

- 🎮 流畅的游戏体验
- 🎨 简洁美观的界面设计
- 📱 支持键盘和触屏操作
- 🏆 分数记录和排行榜
- ⚙️ 可调节的游戏难度

## 🎯 游戏玩法

### 基本规则

1. **控制蛇移动**：使用方向键或触屏滑动控制蛇的前进方向
2. **吃掉食物**：蛇头碰到食物时，蛇身增长一节，分数增加
3. **避免碰撞**：
   - 不要撞到墙壁
   - 不要撞到自己的身体
4. **游戏结束**：发生碰撞时游戏结束，记录最终分数

### 得分系统

| 行为 | 得分 |
|------|------|
| 吃掉普通食物 | +10 分 |
| 连续吃食物（连击） | 额外奖励分 |
| 生存时间奖励 | 每 30 秒 +5 分 |

### 难度等级

| 等级 | 蛇速 | 适合人群 |
|------|------|---------|
| 🟢 简单 | 慢速 | 新手、休闲玩家 |
| 🟡 普通 | 中速 | 有经验玩家 |
| 🔴 困难 | 快速 | 高手挑战 |
| 🟣 地狱 | 极速 | 极限玩家 |

## 🚀 快速开始

### 方式一：在线游玩

访问游戏页面直接开始游玩（如有部署）。

### 方式二：本地运行

```bash
# 克隆项目
cd projects/snake-game

# 打开游戏文件
# 浏览器打开 index.html 即可开始游戏
```

### 方式三：命令行启动

```bash
# 如有 Python 版本
python snake.py

# 如有 Node.js 版本
npm start
```

## 🎮 操作说明

### 键盘操作

| 按键 | 功能 |
|------|------|
| ↑ / W | 向上移动 |
| ↓ / S | 向下移动 |
| ← / A | 向左移动 |
| → / D | 向右移动 |
| 空格 | 暂停/继续 |
| R | 重新开始 |
| M | 静音/取消静音 |

### 触屏操作

| 手势 | 功能 |
|------|------|
| 向上滑动 | 向上移动 |
| 向下滑动 | 向下移动 |
| 向左滑动 | 向左移动 |
| 向右滑动 | 向右移动 |
| 双击 | 暂停/继续 |

## 🛠️ 技术栈
