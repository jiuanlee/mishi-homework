# 贪吃蛇游戏测试计划

## 测试概述

本文档定义了贪吃蛇小游戏的测试用例和测试策略。

## 测试范围

### 1. 基本功能测试

| 测试 ID | 测试项 | 测试步骤 | 预期结果 | 优先级 |
|--------|--------|----------|----------|--------|
| BF-001 | 蛇的移动 | 启动游戏，观察蛇的初始移动 | 蛇应按初始方向匀速移动 | P0 |
| BF-002 | 方向控制 - 上 | 按下上方向键 | 蛇应向上移动 | P0 |
| BF-003 | 方向控制 - 下 | 按下下方向键 | 蛇应向下移动 | P0 |
| BF-004 | 方向控制 - 左 | 按下左方向键 | 蛇应向左移动 | P0 |
| BF-005 | 方向控制 - 右 | 按下右方向键 | 蛇应向右移动 | P0 |
| BF-006 | 反向禁止 | 蛇向右移动时按左键 | 不应立即反向，应忽略无效输入 | P0 |
| BF-007 | 吃食物 | 蛇头移动到食物位置 | 食物被吃掉，蛇身增长一节 | P0 |
| BF-008 | 食物生成 | 食物被吃掉后 | 新食物应在随机位置生成 | P0 |
| BF-009 | 蛇身增长 | 吃食物后 | 蛇身长度应增加 1 节 | P0 |
| BF-010 | 游戏开始 | 启动游戏 | 游戏界面正常显示，蛇在初始位置 | P0 |

| BD-002 | 撞上边界 - 下 | 控制蛇移动到下边界 | 游戏结束，显示失败信息 | P0 |
| BD-003 | 撞上边界 - 左 | 控制蛇移动到左边界 | 游戏结束，显示失败信息 | P0 |

### 3. 计分测试

| 测试 ID | 测试项 | 测试步骤 | 预期结果 | 优先级 |
|--------|--------|----------|----------|--------|

| SC-003 | 连续计分 | 连续吃多个食物 | 分数应累加正确 | P0 |
| SC-004 | 分数显示 | 游戏过程中 | 分数应实时显示在界面上 | P1 |
| SC-005 | 游戏结束分数 | 游戏结束时 | 最终分数应正确显示 | P1 |
| SC-006 | 最高分记录 | 多次游戏后 | 应记录并显示历史最高分 | P2 |

## Bug 等级定义

| 等级 | 描述 | 示例 |
|------|------|------|
| Critical | 导致游戏无法运行或崩溃 | 游戏启动失败、闪退 |
| Major | 核心功能失效 | 蛇无法移动、无法吃食物 |
| Minor | 功能异常但不影响主要流程 | 分数显示错误、图形显示异常 |
| Trivial | 界面或体验问题 | 文字错位、颜色不美观 |
